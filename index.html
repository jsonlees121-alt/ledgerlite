JianJi Privacy Policy

Last Updated: December 2024

1. Data Collection

JianJi is a minimalist personal expense tracking app. This app does not collect any personal user data, does not request any network permissions, and does not communicate with any servers.

2. Data Storage

All expense data you enter (transaction records, profile picture, nickname, settings, etc.) is stored locally on your device using the iOS system's secure storage mechanism (NSUserDefaults). It will not be uploaded to any cloud servers.

3. Data Security

Your data is entirely under your own control. Uninstalling the app will permanently erase all data. We cannot access, read, or back up any of your data.

4. Permission Usage

Permissions requested by this app and their purposes:

  • Photo Library Access: Used only when you actively set a profile picture to select an image. It will not read any other photos.
  • Notifications: Used solely for the daily expense tracking reminder feature. This is activated voluntarily by you and can be turned off at any time.

5. Third-Party Services

This app does not use any third-party analytics, advertising, or tracking services.

6. Children's Privacy

This app is suitable for users of all ages and does not collect any personal information from children.
